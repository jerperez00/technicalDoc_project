Introduccion

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general, que se ha vuelto increíblemente popular debido a su sintaxis clara y legible. Fue creado por Guido van Rossum y se lanzó por primera vez en 1991.

Python es dinámico y soporta múltiples paradigmas de programación, incluyendo procedimental, orientado a objetos e incluso funcional. Es ampliamente utilizado para una variedad de aplicaciones, desde desarrollo web y análisis de datos, hasta inteligencia artificial y aprendizaje automático.

Una de las mayores fortalezas de Python es su comunidad vibrante y activa, que contribuye a un ecosistema robusto de bibliotecas y marcos de trabajo. Esto hace que Python sea una excelente opción para principiantes y expertos por igual.

Python también se destaca por su filosofía de diseño, resumida en el Zen de Python, una colección de 19 “aforismos” que sirven como guía para escribir buen código Python. Algunos de estos principios incluyen: “La legibilidad cuenta”, “Los errores nunca deberían pasar en silencio” y “Debería haber una, y preferiblemente solo una, manera obvia de hacerlo”.

Instalacion

Windows

  1. Ve al sitio web oficial de Python en https://www.python.org/
  2. Haz clic en la sección de descargas y selecciona la versión de Python que deseas instalar. Para la mayoría de los usuarios, la última versión será la mejor opción.
  3. Descarga el instalador de Python y ejecútalo.
  4. Asegúrate de marcar la casilla “Add Python to PATH” antes de hacer clic en “Install Now”. Esto te permitirá ejecutar Python desde cualquier terminal.

macOS

  1. Abre una terminal.
  2. Si no tienes Homebrew, puedes instalarlo con el siguiente comando: "/bin/bash -c $(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  3. Una vez que tienes Homebrew, puedes instalar Python con el comando: brew install python

Linux

Muchas distribuciones de Linux vienen con Python preinstalado. Puedes verificar tu versión de Python con el comando python --version o python3 --version. Si necesitas instalar o actualizar Python, puedes usar el gestor de paquetes de tu distribución. Por ejemplo, en Ubuntu puedes usar apt:

  1. Abre una terminal
  2. Actualiza el índice de paquetes con el comando: sudo apt update
  3. Instala Python con el comando: sudo apt install python3

Ahora deberías tener Python instalado en tu sistema. Recuerda que siempre puedes verificar tu instalación ejecutando python --version en la terminal para ver la versión de Python que tienes instalada.

Sintaxis basica

Indentacion

Python utiliza la indentación para delimitar bloques de código. A diferencia de otros lenguajes que utilizan llaves {} o palabras clave como begin y end,Python utiliza la indentación para determinar el inicio y el fin de los bloques de código. Por ejemplo:

                        
                            if 5 > 2:
                            print("Cinco es mayor que dos")

Variables

En Python, las variables se crean cuando se les asigna un valor por primera vez. No es necesario declarar el tipo de la variable. Por ejemplo:

                        
                            x = 5
                            y = "Hola, mundo"

Comentarios

Los comentarios en Python comienzan con un #. Todo lo que sigue al # en la misma línea es considerado un comentario.

                        
                            # Esto es un comentario
                            print("Hola, mundo") #Esto tambien es un comentario

Operadores

Python soporta una variedad de operadores, incluyendo operadores aritméticos ( +, -,*,/,**,%,//), operadores de asignacion (=,+=,-=, etc.), operadores lógicos (and,or,not), entre otros.

Estructuras de Control de Flujo

Python soporta las estructuras de control de flujo típicas, incluyendo if, elif, else para la toma de decisiones, y for y while para los bucles. Por ejemplo: 

                        
                            # Uso de if, elif, else
                            x = 5
                            if x > 0:
                            print("x es positivo")
                            if x < 0:  
                            print("x es negativo")
                            elif x < 0:
                            print("x es negativo")
                            else:
                            print("x es cero")


                            # Uso de for
                            for i in range (5):
                            print(i)


                            # Uso de while
                            while i < 5:
                            print(i)
                            i +=  1
Tipos de datos

Python tiene varios tipos de datos estándar que se utilizan para definir las operaciones posibles de ellos y el método de almacenamiento.

Números

Python soporta diferentes tipos numéricos:

  • int: Enteros positivos o negativos sin un punto decimal.
  • float: Número real con un punto decimal.
  • complex: Números complejos, se definen utilizando la sintaxis x + yj.

                        x = 10 # int
                        y = 10.5 # float 
                        z = 1j # complex

Cadenas de texto (str)

Las cadenas de texto en Python se crean utilizando comillas simples o dobles. Python también soporta cadenas multilínea.


                        s = "Hola, mundo"

Listas

Las listas son colecciones ordenadas y modificables de elementos. Los elementos de una lista están indexados y se permite la duplicación de elementos.


                        lista = [1, 2, 3, 4, 5]

Tuplas

Las tuplas son colecciones ordenadas e inmutables de elementos. Al igual que las listas, los elementos de una tupla están indexados y se permite la duplicación de elementos.


                        tupla = (1, 2, 3, 4, 5)

Conjuntos

Los conjuntos son colecciones no ordenadas de elementos únicos.


                        conjunto = {1, 2, 3, 4, 5}

Diccionarios

Los diccionarios son colecciones no ordenadas de pares clave-valor.


                        diccionario = {
                            "nombre": "Juan",
                            "edad" : 30
                        }
Funciones y metodos

Definición de funciones

Las funciones en Python se definen usando la palabra clave def, seguida del nombre de la función, paréntesis (), y dos puntos :. Los argumentos de la función se colocan dentro de estos paréntesis. El cuerpo de la función comienza en la siguiente línea, y debe estar indentado.

Aquí tienes un ejemplo de una función simple:


                        def saludo(nombre):
                            return "Hola, " + nombre

Esta función toma un parámetro (nombre) y devuelve un saludo personalizado. Puedes llamar a esta función de la siguiente manera:


                        print(saludo("Mundo")) #Imprime: Hola, Mundo

Funciones incorporadas

Python tiene muchas funciones incorporadas que están siempre disponibles. Aquí tienes algunos ejemplos:

  • print(): Imprime el argumento especificado a la salida estándar (o a la secuencia de texto del archivo).
  • len(): Devuelve el número de elementos en un objeto.
  • type(): Devuelve el tipo de un objeto.
  • str(),int(),float(): Convierten un valor a una cadena de texto, un entero o un número de punto flotante, respectivamente.
  • range(): Genera una secuencia de números.

Estos son solo algunos ejemplos de las funciones y métodos que Python soporta. Cada tipo de datos tiene una serie de métodos que permiten manipular los datos de formas útiles y eficientes.
